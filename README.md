# 🎯 AI 技能库控制台 | Agency Agents

> 212 个即插即用的 AI 专家技能库控制台

一个功能完整的 AI 技能（Agent）管理仪表盘，包含 **212 个即插即用专家**，覆盖开发、设计、营销、数据分析、金融、法律等数十个领域。

## 快速启动

```bash
# 用任何 HTTP 服务器打开
python3 -m http.server 8080
```

然后浏览器访问 `http://localhost:8080` 即可。

## 功能特点

- 🔍 **智能搜索** — 实时筛选技能和智能体
- 🔖 **分类筛选** — 按类别/标签快速定位
- 📋 **一键复制** — 点击即可复制使用命令
- 🌐 **中文界面** — 全中文操作体验
- 🎨 **毛玻璃设计** — 现代化暗色 UI，流畅交互
- 📱 **响应式布局** — 桌面和移动端均可使用
